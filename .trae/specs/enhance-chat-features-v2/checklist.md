# Checklist

## 第一阶段：消息功能增强

- [x] 消息引用/回复：可引用消息进行回复
- [x] 消息引用/回复：引用消息显示预览
- [x] 消息转发：可转发消息到其他会话
- [x] 消息收藏：可收藏/取消收藏消息
- [x] 消息收藏：可查看收藏列表
- [x] @提及：群聊可 @成员
- [x] @提及：被 @者收到特别通知
- [x] 表情包选择器：可选择并插入表情
- [x] 消息发送状态：显示发送中/失败状态
- [x] 消息发送状态：发送失败可重发

## 第二阶段：群组功能增强

- [x] 群公告：群主/管理员可发布公告
- [x] 群公告：群成员可查看公告
- [x] 群头像：可上传群头像
- [x] 群管理员：群主可设置管理员
- [x] 群管理员：管理员有相应权限
- [x] 群邀请链接：可生成邀请链接
- [x] 群邀请链接：可通过链接加入群组

## 第三阶段：社交功能增强

- [x] 好友备注名：可设置好友备注名
- [x] 好友备注名：备注名正确显示
- [x] 好友分组：可创建/删除分组
- [x] 好友分组：可将好友移入分组
- [x] 用户个性签名：可设置个性签名
- [x] 在线状态自定义：可设置在线/忙碌/离开/隐身
- [x] 黑名单：可拉黑用户
- [x] 黑名单：拉黑后无法收到对方消息

## 第四阶段：用户体验优化

- [x] 会话置顶：可置顶/取消置顶会话
- [x] 会话置顶：置顶会话显示在顶部
- [x] 消息免打扰：可设置会话免打扰
- [x] 消息免打扰：免打扰会话不发送通知
- [x] 会话删除：可删除会话
- [x] 好友删除：可删除好友
- [x] 上拉加载更多：可加载历史消息
- [x] 图片预览：点击图片可全屏预览
- [x] 图片预览：支持缩放和滑动
