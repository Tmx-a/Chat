# Checklist

## 第一阶段：消息布局问题

- [x] 对方消息之间有足够的垂直间距（至少16px）
- [x] 连续消息间距合理，视觉上清晰可辨
- [x] 自己的消息在大屏幕上靠右对齐，不居中
- [x] 消息气泡宽度合理（不超过500px）

## 第二阶段：已读/未读提示问题

- [x] 已读/未读提示清晰显示在消息气泡下方
- [x] 已读/未读提示不被任何元素遮挡
- [x] 发送状态图标（转圈、错误）正确显示

## 第三阶段：输入框问题

- [x] 输入框中的文字垂直居中显示
- [x] 输入框上方无多余空白
- [x] 多行输入时文字位置正确

## 第四阶段：消息发送问题

- [x] 发送一条消息只显示一条消息
- [x] 发送成功后转圈图标消失
- [x] 发送失败时显示错误图标并可重试
- [x] 已读/未读状态正确显示
